**Goal:** Fix FAQ context loss by (1) passing conversation history to LLMs and (2) adding WhatsApp reply buttons as fallback when the bot can't find an answer.

## Task 1: Pass conversation history to both LLMs

**Files:**
- `n8n/workflows/whatsapp-chatbot.json`

  - Node "Extract Message Fields": add `button_id` assignment, update `message_text` to handle `buttonsResponseMessage`
  - Node "Merge Conversation Data": map button IDs to natural language text
  - Node "Format FAQs for LLM": add `conversation_history` field formatted from `recent_messages`
  - Node "Check FAQ Triggers Handoff": set `send_buttons: true` instead of `human_handoff` when FAQ fails
- `n8n/workflows/sub-classify-intent.json`

  - Node "Set FAQ Answer Output": fix `is_handoff` detection to check for `NEEDS_CLARIFICATION` sentinel
