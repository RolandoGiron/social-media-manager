           cr.id AS recipient_id,

           p.last_name,

           .replace(/\{\{nombre\}\}/g, item.first_name || '')

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan | Severity | Status |
|-----------|----------|-----------|-------------|-----------------|----------|--------|
| T-05-09 | Elevation of Privilege | webhook-trigger | mitigate | Webhook path `campaign-blast` only routes through internal Docker DNS (`n8n:5678`). Caddy reverse proxy from Phase 1 does not include a public route for `/webhook/campaign-blast`. Verified manually in checkpoint Task 3. | high | mitigated |
| T-05-10 | Denial of Service / WhatsApp ban | send-whatsapp loop | mitigate | Rate limiting via `compute-jitter` + `wait-jitter` enforces 3000-8000 ms between sends (D-08, WA-02). Implemented as `Math.floor(Math.random() * 5000) + 3000`. | high | mitigated |
| T-05-11 | Tampering / Race | finalize-status | mitigate | Final UPDATE uses `CASE WHEN status = 'cancelled' THEN 'cancelled' ELSE 'completed' END` so a mid-loop cancellation cannot be silently overwritten when the splitInBatches `done` branch fires (RESEARCH Pitfall 3). | high | mitigated |
| T-05-12 | Tampering | update-failed error_message string interpolation | mitigate | Error message is escaped (`replace(/'/g, "''")`) and truncated to 500 chars before being interpolated into SQL.
