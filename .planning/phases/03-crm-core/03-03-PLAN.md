       ```python
       import streamlit as st
       from components.sidebar import render_sidebar
       from components.database import fetch_templates, insert_template, delete_template

       ```

    2. Session state initialization:
       - `st.session_state.setdefault("plantillas_mode", "list")` — values: "list" or "editor"
       - `st.session_state.setdefault("edit_template_id", None)` — None for new, UUID for editing

    3. Page title:
       ```python
       st.title("Plantillas de Mensaje")
       ```

    4. **Editor mode** (when `plantillas_mode == "editor"`), per D-09:
       - Two equal columns: `col_editor, col_preview = st.columns([1, 1])`
       - **Left column (Editor):**
         - `template_name = st.text_input("Nombre de plantilla", placeholder="Ej: Promocion mensual, Recordatorio de cita...", key="tpl_name")`
         - `category = st.selectbox("Categoria", options=["general", "promocion", "recordatorio"], key="tpl_category")` (per D-11)
         - `body = st.text_area("Cuerpo del mensaje", placeholder="Escribe tu mensaje. Usa {{nombre}}, {{fecha}} u otras variables entre llaves dobles.", height=300, key="tpl_body")`
         - Two buttons in a row:
           - `if st.button("Guardar plantilla", type="primary"):`
             - Validate: if not template_name.strip(): `st.error("El nombre de la plantilla es obligatorio.")` and stop
             - Validate: if not body.strip(): `st.error("El cuerpo del mensaje no puede estar vacio.")` and stop
             - Extract variables: `variables = extract_variables(body)` (per D-10)
             - Insert: `insert_template(template_name.strip(), body.strip(), variables, category)`
             - `st.success("Plantilla guardada exitosamente.")`
             - Reset mode to "list", rerun
           - `if st.button("Descartar cambios"):`
             - Reset mode to "list", rerun

       - **Right column (Preview):** per D-09 live side-by-side preview
         - `st.subheader("Vista previa")`
         - If body is not empty:
           - `rendered = render_preview(body)`
           - Display rendered text in a container with `st.markdown(rendered)` inside `st.container(border=True)`
           - Below preview, show detected variables:
             - `vars_found = extract_variables(body)`
             - If vars_found: `st.caption(f"Variables detectadas: {', '.join(vars_found)}")`
             - If not vars_found and body.strip(): `st.caption("No se detectaron variables.")`
         - If body is empty:
           - `st.info("Escribe un mensaje en el editor para ver la vista previa.")`

    5. **List mode** (when `plantillas_mode == "list"`):
       - "Nueva plantilla" button:
         ```python
         if st.button("Nueva plantilla", icon=":material/add:"):
             st.session_state.plantillas_mode = "editor"
             st.session_state.edit_template_id = None
             st.rerun()
         ```
       - Fetch templates: `templates = fetch_templates(active_only=True)`
       - If no templates: show empty state per UI-SPEC
         - `st.info("No hay plantillas creadas")`
         - `st.markdown("Crea tu primera plantilla de mensaje para usarla en campanas de WhatsApp.")`
       - If templates exist:
         - Build display DataFrame with columns: Nombre (name), Categoria (category), Variables (join variables list with ", "), Creada (created_at formatted as "01 abr 2026")
         - Display with `st.dataframe(display_df, use_container_width=True, hide_index=True)`
         - Below the list, for each template show a delete option:
           - Use `st.expander(f"Acciones: {t['name']}")` for each template
           - Inside: `st.warning(f"Eliminar plantilla: Estas seguro de que deseas eliminar '{t['name']}'? Esta accion no se puede deshacer.")`
           - `if st.button("Eliminar", key=f"del_{t['id']}"):`
             - `delete_template(str(t["id"]))`
             - `st.success("Plantilla eliminada.")`
             - `st.rerun()`

    **Important implementation notes:**
    - The live preview updates on every keystroke in the body text_area because Streamlit reruns on every widget change. No debouncing needed.
    - Use try/except around DB calls with st.error() fallback.
    - Template date formatting for "Creada" column: use `t["created_at"].strftime("%d %b %Y")` if datetime, or parse string if needed. Months in Spanish can be handled with a small dict mapping or locale — simplest is a dict: `{"Jan": "ene", "Feb": "feb", "Mar": "mar", "Apr": "abr", "May": "may", "Jun": "jun", "Jul": "jul", "Aug": "ago", "Sep": "sep", "Oct": "oct", "Nov": "nov", "Dec": "dic"}`.
